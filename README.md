# Despensa del Cuartel

App de gestion de inventario de despensa con interfaz radial interactiva.

## Descripcion

Aplicacion Android para gestionar el inventario de tu despensa de manera intuitiva.
Organiza productos por categorias usando una rueda radial visual.

## Features

- Rueda Radial Interactiva - Navega por categorias tocando las secciones
- Anadir Productos - Agrega nuevos productos con un tap
- Control de Stock - Gestiona cantidades minimas y maximas
- Sincronizacion Firestore - Datos siempre actualizados en la nube
- Modo Oscuro - Interfaz adaptada para cualquier condicion de luz

## Categorias

- Frutas
- Carnes
- Pan
- Cafe
- Lacteos
- Medicamentos
- Cerveza
- Verduras
- Enlatados

## Tech Stack

- Kotlin + Jetpack Compose
- Material Design 3
- Firebase Firestore
- MVVM Architecture
